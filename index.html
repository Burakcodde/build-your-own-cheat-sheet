HTML & CSS Cheatsheet

HTML Temel Etiketleri

Etiket Açıklama Örnek
<html> Kök etiket. <html></html>
<head> Başlık bilgileri. <head><title>Başlık</title></head>
<body> Gövde içeriği. <body>İçerik</body>
<h1> - <h6> Başlık etiketleri (seviye 1-6). <h1>Başlık</h1>
<p> Paragraf etiketi. <p>Paragraf metni</p>
<a> Bağlantı etiketi. <a href="https://www.google.com">Google</a>
<img> Resim etiketi. <img src="resim.jpg" alt="Resim Açıklaması">

CSS Temel Özellikleri

Özellik Açıklama Örnek
`color` Metin rengi. `color: red;`
`background-color` Arka plan rengi. `background-color: #f0f0f0;`
`font-size` Yazı boyutu. `font-size: 16px;`
`font-family` Yazı tipi. `font-family: Arial, sans-serif;`
`margin` Dış boşluk. `margin: 10px;`
`padding` İç boşluk. `padding: 5px;`
`border` Kenarlık. `border: 1px solid black;`
`text-align` Metin hizalama. `text-align: center;`
